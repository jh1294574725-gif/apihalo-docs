# OpenClaw

更新时间：2026-04-15

如果你的 OpenClaw 版本支持 `OpenAI` 或 `OpenAI Compatible` 自定义网关，可按下面填写。

## 填写方式

- Provider：`OpenAI` / `OpenAI Compatible` / `Custom`
- Base URL：`https://apihalo.com/v1`
- API Key：你的 ApiHalo Key
- Model：从 `GET /v1/models` 返回结果中选择

## 常见问题

如果客户端会自动补 `/v1`，则 Base URL 改填：

```text
https://apihalo.com
```

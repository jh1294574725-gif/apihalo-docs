# 其他工具

更新时间：2026-04-15

如果某个工具没有单独列在本目录，但它支持 `OpenAI`、`OpenAI Compatible`、`Custom OpenAI` 或 `OpenAI-style`，通常都可以按同一套方式接入。

## 通用填写模板

- Provider：`OpenAI` / `OpenAI Compatible`
- Base URL：`https://apihalo.com/v1`
- API Key：你的 ApiHalo Key
- Model：从 `/v1/models` 返回结果中选择

## 通用排查顺序

1. 先确认工具是否支持自定义 Base URL
2. 再确认是否自动拼接 `/v1`
3. 再确认模型名是否来自 `/v1/models`
4. 最后再排查是否是模型能力未开通

## 最稳的建议

如果某个工具接入失败，优先先用 `curl` 或 OpenAI SDK 验证连通性。

只要 `curl` 能通，绝大多数支持 OpenAI Compatible 的工具都能接进去。

# Claude Code

更新时间：2026-04-15

如果你的 Claude Code 版本支持自定义 `OpenAI Compatible` 网关，可按本页接入。

## 建议填写

- Provider：`OpenAI Compatible` 或 `Custom OpenAI`
- Base URL：`https://apihalo.com/v1`
- API Key：你的 ApiHalo Key
- Model：使用 `/v1/models` 返回的模型 ID

## 使用建议

- 如果只是为了稳定接入，优先选择 OpenAI 兼容模式
- 如果当前版本不支持自定义 OpenAI 网关，请改用支持 OpenAI Compatible 的客户端或适配层

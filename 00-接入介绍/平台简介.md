# 平台简介

更新时间：2026-04-15

ApiHalo 是面向开发者的统一 AI API 接入平台，网站地址：

`https://apihalo.com/`

你在业务里真正对接的对象是 ApiHalo，而不是某个单独的上游平台。

这意味着：

- 你使用的是 ApiHalo 的 `API Key`
- 你调用的是 ApiHalo 的 `Base URL`
- 你能用哪些模型，以 ApiHalo 返回结果为准

## 公开接入原则

为了保证你后续即使更换上游渠道也尽量不用重写代码，ApiHalo 对外默认统一采用 `OpenAI Compatible` 接口口径。

对于绝大多数客户，最推荐的接入方式只有两步：

1. `GET https://apihalo.com/v1/models`
2. `POST https://apihalo.com/v1/chat/completions`

## 适用范围

本文档适用于：

- 自己写代码接入
- 使用 OpenAI SDK
- 使用支持 `OpenAI` 或 `OpenAI Compatible` 的第三方工具
- 接入文本、代码、多模态、向量、图像、音频、视频等能力

## 你只需要记住三件事

1. 先拉取模型列表，再决定调用哪个模型
2. 不要把上游平台公开模型总数，理解为你当前账号真实可用模型总数
3. 高级能力按模型开放，不要默认所有模型都支持同样的参数和接口

# ApiHalo 文档目录

更新时间：2026-04-15

请先记住 3 条规则：

1. 客户可用模型始终以 `GET https://apihalo.com/v1/models` 返回结果为准
2. 客户永远使用 ApiHalo 的 `API Key` 和 `Base URL`，不要直接套用其他平台的接入参数
3. 默认优先按 `OpenAI Compatible` 文档接入

## 补充说明

- 如果你的客户端只支持 `OpenAI` 或 `OpenAI Compatible`，优先看 [OpenAI格式（支持各大原厂模型）](01-AI大模型API/OpenAI格式（支持各大原厂模型）.md)

- 如果模型后来新增、下架或替换，请重新拉取一次 `GET /v1/models`

# 豆包系列-绘画

更新时间：2026-04-15

本页适用于已开通豆包/火山系图像生成能力的账号。

## 推荐地址

```text
POST https://apihalo.com/v1/images/generations
```

## 请求示例

```bash
curl https://apihalo.com/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "YOUR_DOUBAO_IMAGE_MODEL",
    "prompt": "一只坐在窗边看雨的橘猫"
  }'
```

## 说明

- 这是面向图像生成能力的标准公开入口
- 如果你的模型支持尺寸、风格、负面提示词等扩展参数，可以在已开通前提下按模型能力追加
- 如果你不确定该模型是否属于图片生成能力，请先看 `/v1/models`

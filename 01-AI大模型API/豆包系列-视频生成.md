# 豆包系列-视频生成

更新时间：2026-04-15

本页适用于已开通豆包/火山系专项视频生成能力的账号。

## 推荐地址

```text
POST https://apihalo.com/v1/video/generations
```

## 请求示例

```bash
curl https://apihalo.com/v1/video/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "YOUR_DOUBAO_VIDEO_MODEL",
    "prompt": "宇航员在月球上漫步",
    "duration": 5,
    "width": 1280,
    "height": 720
  }'
```

## 查询任务

```bash
curl https://apihalo.com/v1/video/generations/YOUR_TASK_ID \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 说明

- 豆包视频能力建议用本页专项接口
- 如果你调用的是统一 OpenAI 视频能力，也可参考 [通用视频生成API](通用视频生成API.md)
- 具体开放模型以 `/v1/models` 和账号能力为准

# 查询账户信息

更新时间：2026-04-15

ApiHalo 提供 OpenAI 兼容风格的账户信息与用量查询接口，可用于调用前自检、控制台展示或业务侧额度提示。

## 一、查询账户额度

```bash
curl https://apihalo.com/v1/dashboard/billing/subscription \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 二、查询已用量

```bash
curl https://apihalo.com/v1/dashboard/billing/usage \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 三、适用场景

- 客户端显示剩余额度
- 控制台展示账户使用情况
- 调用前自检 Key 是否可用

## 四、注意事项

- 额度单位以站点展示口径为准
- 不同部署可能按金额、额度或 tokens 展示
- 如果只关心“能不能调用”，这两个接口属于辅助接口，不是主业务接口

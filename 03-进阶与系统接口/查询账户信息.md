# 查询账户信息

更新时间：2026-04-15

ApiHalo 提供 OpenAI 兼容风格的账户信息与用量查询接口。

## 查询账户额度

```bash
curl https://apihalo.com/v1/dashboard/billing/subscription \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 查询已用量

```bash
curl https://apihalo.com/v1/dashboard/billing/usage \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 适用场景

- 客户端显示剩余额度
- 控制台展示账户使用情况
- 接入前自检 Key 是否可用

## 注意事项

- 额度单位以站点展示口径为准
- 不同部署可能按金额、额度或 tokens 展示
- 如果你只关心“能不能调用”，可把这两个接口当作辅助接口，不是主业务接口
